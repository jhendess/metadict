Basic querying

Metadict allows you to query multiple dictionaries from different search engines at once. Click settings in the top-right corner select the dictionaries you want to search in. Enter your query and hit search.

Registration

You can use all features of Metadict without registering a user account. However, there are planned features which will let you benefit from a registered user account - this includes vocabulary trainers and social interaction with other users! Click "Register / Login" and create your own - no personal data necessary.

Using Metadict on mobile devices

You can use Metadict just like any other webapp on most mobile devices. Check below for your operating system.

Android
  1. Open the Metadict page in your web browser
  2. Open your browser's menu
  3. Select "Add to Home screen" to create an icon
iOS
  1. Open the Metadict page in Safari
  2. Click the share button
  3. Select "Add to Home screen" to create an icon
